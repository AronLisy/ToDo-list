What are your tasks for today?


    Welcome to a task list. This is the main part of this page.

    Are you sometimes overwhelmed by everything you need to do? Write down every little task and after each one press SAVE button or enter.

    Once you copleted a task, simply click DONE button and the task will disappear from your life.

    All tasks are saved in a local storage, so even if you exit web browser and than open this page again, they will be still here for you.

    Alarms

    • 00:00:00 #

    This is an alarm list. By default there are no alarms.

    By clicking ADD NEW ALARM button, you can add as many alarms you like. After creating new alarm, set hours, minutes and seconds. You can also add a message into input field. For example: "Take a break!" or "Get back to work!"

    When the time is up, you will recieve message you wrote previously in a window and alarm sound will ring.

    Alarms are not saved in local storage, therefore they are reset each time you close or refresh page.
